###Key features
* Offline caching
* Dictionary with examples
* Voice input
* Listen how sounds the word or sentence in other language
* History
* Favorites

###Libraries
* RxJava 2
* Dagger 2
* Retrofit 2 and Retrofit Mock mode for debug builds
* Butterknife
* Retrolambda
* Stetho
* Cupboard
* Speechkit Yandex
